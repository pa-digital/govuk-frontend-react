<h2>Content</h2>

<p>
  The content tag is used to render standard typography with paragraph blocks
  styled using GDS styling.
</p>

<p>
  You can view the full implementation details on the Typography section of the
  GDS website.
</p>

<a href="https://design-system.service.gov.uk/styles/typography/" target="_blank" rel="noreferrer">
  Learn more >>
</a>

<h2>Options</h2>

<p>
  The content block renders in a p tag as standard but can be overridden to
  render as a pre tag.
</p>
