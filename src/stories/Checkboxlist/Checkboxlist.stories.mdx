<h2>Options</h2>

<p>The Checkbox list can be configured using the following:</p>

<ul>
  <li>Label - contains the element label</li>
  <li>Pre Checked - allows checkboxes to be pre-checked</li>
  <li>Compact - reduces the overall size of the checkboxes</li>
  <li>
    MultiQuestion - adjusts the label to not contain an H1 tag to be compliant
  </li>
  <li>
    Divider - provides a divider in the checkbox list - defaults to "or" but can
    be overridden
  </li>
  <li>Hints - contains hints for the overall checkox and each item</li>
  <li>Error - displays the error and configures all aria attributes</li>
</ul>
