<h2>Options</h2>

<p>The Checkbox list can be configured using the following:</p>

<ul>
  <li>
    Identifier - main identifier for the checkboxlist - builds all subsequent
    element id's
  </li>

  <li>Header - contains the checkbox list header</li>
  <li>Data - the array of CheckBoxDataProps that contains the checkboxes</li>

  <li>
    Hint (Optional) - contains a hint for the overall checkox and each item
  </li>

  <li>Compact (Optional) - reduces the overall size of the checkboxes</li>

  <li>
    MultiQuestion (Optional) - adjusts the label to not contain an H1 tag to be
    compliant
  </li>

  <li>
    ShowToggle (Optional) - displays a toggle option that can select/de-select
    all boxes
  </li>

  <li>
    Error (Optional) - displays the error and configures all aria attributes
  </li>

  <li>
    <b>onValueChange</b> - returns an array of CheckBoxDataProps
  </li>
</ul>

<p>For each checkbox within the data element</p>

<ul>
  <li>Value - the value assigned with the checkbox</li>
  <li>Text - the text displayed in the checkbox label</li>
  <li>Checked (Optional) - Determines if the checkbox is checked intitially</li>
  <li>Hint (Optional) - Displays a hint for the checkbox</li>
  <li>Divider (Optional) - Makes the element display as a divider</li>

  <li>
    Exclusive (Optional) - Makes the checkbox exclusive and will un-check all
    others when selected
  </li>
</ul>
