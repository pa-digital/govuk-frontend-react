<h1>GOV.UK Design System Front End React Library</h1>

<p>This package is based on the GOV.UK Design System referred to as GDS</p>

<p>
  This library has been designed to be used as an npm package for react
  projects.
</p>

<p>It provides out-of-the-box GDS compliant components.</p>

<p>
  It is based on the govuk-frontend npm package with component html drawn from
  the GDS component site itself.
</p>

<p>
  Each component is grouped where necessary and has an Information page and
  Implementation page within the Storybook.
</p>
