<h2>Navigation</h2>

<p>
  A breakdown of some of the more commonly used GDS components for navigation.
</p>

<h3>Back Link</h3>
<p>
  Use the back link component to help users go back to the previous page in a
  multi-page transaction.
</p>

<p>
  You can view the full implementation details on the{' '}
  <a
    href="https://design-system.service.gov.uk/components/back-link/"
    target="_blank"
    rel="noreferrer"
  >
    Back Link
  </a>{' '}
  section of the GDS website.
</p>
<hr />
<h3>Breadcrumbs</h3>

<p>
  The breadcrumbs component helps users to understand where they are within a
  website's structure and move between levels.
</p>

<p>
  You can view the full implementation details on the{' '}
  <a
    href="https://design-system.service.gov.uk/components/breadcrumbs/"
    target="_blank"
    rel="noreferrer"
  >
    Breadcrumbs
  </a>{' '}
  section of the GDS website.
</p>
<h4>Options</h4>
<p>The breadcrumb can be configured using the following:</p>

<ul>
  <li>Standard - a standard implementation of the breadcrumb links</li>
  <li>
    CollapseOnMobile - reduces the size of the breadcrumb on mobile to display
    only the first and last link
  </li>
</ul>
<hr />
<h3>Nav Link</h3>

<p>
  Whilst not a specific GDS Component.This implements the links to both internal
  and external sites as specified on the{' '}
  <a
    href="https://design-system.service.gov.uk/styles/typography/#links"
    target="_blank"
    rel="noreferrer"
  >
    Typography
  </a>{' '}
  section of the GDS website.
</p>
<h4>Options</h4>
<p>The Nav link can be configured using the following:</p>

<ul>
  <li>Default - uses the React Router internal routing enging</li>
  <li>
    External - implements an external link with all appropriate attributes
  </li>
  <li>
    Assistive Text - Adds aria-label where the link text is duplicated or
    requires explanation
  </li>
</ul>
<hr />
<h3>Header Navigation</h3>

<p>
  The header navigation component implements the navigation component within a
  header on a GDS styled website.
</p>
<hr />
<h3>Detached Navigation</h3>

<p>
  The detached navigation component implements the navigation within a
  nav element using the same functionality as the header navigation on a GDS styled website.
</p>
