<h3>Breadcrumbs</h3>

<p>
  The breadcrumbs component helps users to understand where they are within a
  website's structure and move between levels.
</p>

<p>
  You can view the full implementation details on the Breadcrumbs section of the
  GDS website.
</p>

<a
  href="https://design-system.service.gov.uk/components/breadcrumbs/"
  target="_blank"
  rel="noreferrer"
>
  Learn more >>
</a>

<h4>Options</h4>
<p>The breadcrumb can be configured using the following:</p>

<ul>
  <li>Standard - a standard implementation of the breadcrumb links</li>

  <li>
    CollapseOnMobile - reduces the size of the breadcrumb on mobile to display
    only the first and last link
  </li>
</ul>

<hr />

<h3>Nav Link</h3>

<p>
  Whilst not a specific GDS Component.This implements the links to both internal
  and external sites as specified on the Typography section of the GDS website.
</p>

<a
  href="https://design-system.service.gov.uk/styles/typography/#links"
  target="_blank"
  rel="noreferrer"
>
  Typography
</a>

<h4>Options</h4>
<p>The Nav link can be configured using the following:</p>

<ul>
  <li>Default - uses the React Router internal routing enging</li>

{' '}

<li>External - implements an external link with all appropriate attributes</li>

  <li>
    Assistive Text - Adds aria-label where the link text is duplicated or
    requires explanation
  </li>
</ul>
