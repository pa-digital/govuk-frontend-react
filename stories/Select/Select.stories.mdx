<h2>Select</h2>

<p>
  The select component should only be used as a last resort in public-facing
  services because research shows that some users find selects very difficult to
  use.
</p>

<p>
  You can view the full implementation details on the{' '}

    target="_blank"
    rel="noreferrer"
  >
    select
  </a>{' '}
  section of the GDS website.
</p>
<h2>Options</h2>

<p>The select component can be configured using the following:</p>

<ul>
  <li>Value - allows a particular option to be pre-selected</li>
  <li>
    MultiQuestion - adjusts the label to not contain an H1 tag to be compliant
  </li>
  <li>Hint - contains hints for the overall checkox and each item</li>
  <li>Error - displays the error and configures all aria attributes</li>
</ul>
